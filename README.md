# proyectofinalprogramacion1
proyecto recetario
Proyecto para programacion 1 de la carrera de TÃ©cnico en Desarrollo de Software
El presente trabajo es sobre un recetario, se permite las operaciones basicas de abm
Se utiliza en la pantalla principal los botones de buscar, nueva receta, modificar receta
eliminar receta, refrescar
un treeview donde se muestran todas las recetas
al presionar el boton de nueva receta se abre una ventana top level donde se ingresan los datos
al precionar guardar, se limpian todos los campos y se guarda lo ingresado en un archivo csv con el metodo "a"
al presionar modificar receta se abre una ventana con los campos llenos de la receta seleccionada en el treeview
se modifican todos los campos, al presionar guardar se registran los cambios en el archivo csv
el boton eliminar elimina el elemento seleccionado en el treeview
el boton refrescar actualiza el treeview mostrando los cambios que se realizaron en las ventanas nueva receta y modificar receta
